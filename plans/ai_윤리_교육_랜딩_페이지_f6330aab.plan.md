# AI 윤리 교육 랜딩 페이지

제작 계획

## 개요

기존 골든리트리버 페이지를 AI 윤리 교육 자료 Repository 소개 랜딩 페이지로 전면 교체합니다. 모바일 우선 설계로 밝은 톤의 현대적인 UI를 구현합니다.

## 파일 구조

- `index.html`: 메인 HTML 구조

- `style.css`: 밝은 톤의 모바일 최적화 스타일
- `script.js`: 문의폼 제출 처리 (콘솔 출력)

## 주요 섹션 구성

### 1. 히어로 섹션

- 타이틀: "인공지능 윤리 교육을 위한 교육 자료 Repository"

- 간단한 소개 문구
- 밝은 그라데이션 배경

### 2. 주제 섹션 (3개)

각 주제를 카드 형태로 표시:

- **주제1**: AI 교육 수업계획안 & 성공 사례

- **주제2**: 청소년과 AI 활용

- **주제3**: 인공지능 윤리 관련 웹사이트

각 카드는 나중에 링크를 추가할 수 있도록 플레이스홀더 구조로 제작

### 3. 커피챗 문의폼

- 성함 및 소속 (input)

- 연락받을 이메일 (input type="email")

- 문의 내용 (textarea)

- 제출 버튼

- 제출 시 콘솔에 데이터 출력

## 디자인 컨셉

- **색상**: 밝은 톤 (화이트, 연한 파스텔 계열)

- **타이포그래피**: 깔끔한 한글 폰트

- **레이아웃**: 모바일 우선, 단일 컬럼

- **애니메이션**: 부드러운 스크롤 효과, 호버 효과

- **카드 디자인**: 그림자와 둥근 모서리로 깔끔한 느낌

## 구현 세부사항

### HTML 구조

- 반응형 메타 태그 포함

- 시맨틱 HTML5 태그 사용 (header, section, form 등)

- 접근성 고려 (label, aria-label 등)

### CSS 스타일

- 모바일 퍼스트 반응형 디자인
- 밝은 그라데이션 배경
- 카드 기반 레이아웃

- 부드러운 트랜지션 효과
- 터치 친화적인 버튼 크기

### JavaScript 기능

- 문의폼 유효성 검사
- 제출 시 콘솔에 JSON 형태로 출력
- 폼 리셋 기능

- 부드러운 스크롤 애니메이션
